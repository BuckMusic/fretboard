# fretboard and tab maker
# This is an excel based spreadsheet program to display the complete fretboard layout of any stringed instrument.
# The program uses a setup page to specify the number of strings, tuning of the strings, number of frets, display of flats/sharps, as well as other specifics.
# The program expands to allow input of a series of notes to display a tab style layout on the specific instrument.
