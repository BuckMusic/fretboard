# Fretboard and tab maker excel spreadsheets.
# These are excel based spreadsheet programs to display the complete fretboard layout of any stringed instrument.
# The programs use a setup page to specify the number of strings, tuning of the strings, number of frets, display of flats/sharps, as well as other specifics etc..
# The programs expand to allow input of a series of notes to display a tab style layout on any spicific instrument.
